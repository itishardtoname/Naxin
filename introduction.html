关于实验室

我们是谁？

“ThoughtWorks实验室”，全称"西安邮电大学联合创新实验室"，是ThoughtWorks公司联合西安邮电大学创立。

我们在干什么？

“ThoughtWorks实验室”，在学校计算机学院芸芸实验室中也是唯一一间以前端为首，后台为辅的实干型实验室。

我们期待什么？

“ThoughtWorks实验室”，期待着有对技术的热爱，积极乐观向上的同学的加入。

前 端 进 阶 技 术

ES6

ES6,主要是为了解决ES5的先天不足,但是目前浏览器的JavaScript是ES5版本，大多数高版本的浏览器也支持 ES6，不过只实现了ES6的部分功能。

jQuery

jQuery是一个JavaScript函数库,还提供了大量的插件。 jQuery库包含以下功能:JavaScript特效和动画,HTML DOM遍历和修改,AJAX和Utilities

Vue.js

Vue 只关注视图层， 采用自底向上增量开发的设计。 Vue 的目标是通过尽可能简单的 API 实现响应的数据绑定和组合的视图组件。

Bootstrap

Bootstrap是目前最受欢迎的前端框架。 Bootstrap 是最受欢迎的前端组件库，用于开发响应式布局、移动设备优先的 WEB 项目。
